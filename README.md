# ⚙ Simulation Logic

The simulation models rocket motion using **basic physics principles**.

### 🧮 Core Equation

The system applies **Newton’s Second Law of Motion**:

F = m × a

Where:

- **F** = Net Force  
- **m** = Mass of rocket  
- **a** = Acceleration  

The rocket acceleration is calculated using:

Acceleration = (Thrust − Drag − Gravity) / Mass

As fuel burns:

- Rocket mass decreases  
- Acceleration increases  
- Altitude changes dynamically

# 🚀 Physics Concepts Demonstrated

## 1️⃣ Newton’s Second Law

This law explains how **force affects motion**.  
In rockets, acceleration occurs when the **thrust force exceeds gravitational and drag forces**.

### Key Insight

Lower rocket mass and higher thrust increase acceleration.

---

## 2️⃣ Thrust

Thrust is the **force generated by rocket engines** when high-speed exhaust gases are expelled downward.

According to **Newton’s Third Law**, the downward force of expelled gases produces an **equal and opposite upward force**, pushing the rocket upward.

### Key Insight

Higher thrust allows rockets to lift heavier payloads and escape Earth's gravity.

---

## 3️⃣ Drag

Drag is the **air resistance opposing rocket motion**.

Drag depends on:

- Rocket speed  
- Air density  
- Rocket shape  

To reduce drag, rockets are designed with **streamlined aerodynamic structures**.

### Key Insight

Lower drag improves rocket efficiency and launch success.

---

## 4️⃣ Payload

Payload refers to the **cargo carried by the rocket**, such as:

- Satellites  
- Scientific instruments  
- Astronauts  

Increasing payload increases the rocket's **mass**, requiring more thrust and fuel.

### Key Insight

Payload weight directly impacts rocket performance and achievable altitude.

## ⚙️ Integration Details

This project integrates **physics-based rocket simulation, interactive UI controls, and mission data visualization** to create an educational space mission analysis platform.

* **Physics Simulation Engine:**  
  The `rocket_simulation()` function performs a **step-by-step numerical simulation** of rocket flight. It calculates acceleration using the equation **F = (Thrust − Drag − Gravity) / Mass**, then integrates acceleration into velocity and velocity into altitude over multiple simulation steps. As fuel burns during the simulation, the rocket's mass gradually decreases, improving acceleration and changing the trajectory.

* **UI-to-Simulation Integration:**  
  Interactive **Streamlit sliders** allow users to modify parameters such as **payload weight, thrust force, drag coefficient, fuel amount, burn rate, and simulation steps**. These values are instantly passed into the simulation function, recalculating the rocket trajectory in real time.

* **State Management & User Interaction:**  
  Streamlit's `st.session_state` is used to manage navigation between pages such as **Splash Screen, Login Page, and Dashboard**. It also stores user information during the session to maintain a smooth interactive experience across reruns.

* **Data Visualization Integration:**  
  The application uses **Plotly Express** to generate interactive charts directly from the dataset and simulation results. Mission data is processed with **Pandas** and visualized through scatter plots, line charts, bar charts, box plots, and correlation heatmaps.

## 🔬 Research Context — Rocket Dynamics & Newton's Law

The simulation and analysis within this project are grounded in **fundamental physics principles used in real-world rocket engineering**.

The primary law demonstrated is **Newton’s Second Law of Motion**, which states:

**F = m × a**

Where:

- **F** = Net Force acting on the rocket  
- **m** = Mass of the rocket  
- **a** = Acceleration  

During a rocket launch:

1. **Thrust** generated by rocket engines pushes the rocket upward.
2. **Gravity** pulls the rocket downward.
3. **Drag** from atmospheric resistance opposes motion.
4. **Payload mass** increases the total mass of the rocket.

The rocket accelerates upward only when:

**Thrust > (Gravity + Drag)**

As fuel burns:

- Rocket mass decreases
- Acceleration increases
- The rocket gains velocity and altitude

This simplified physics model helps users understand how **engineering parameters influence rocket performance and mission outcomes**.
